Instalación

Requisitos

SheShe requiere Python >=3.9. Es mejor trabajar dentro de un entorno virtual. Las dependencias base incluyen numpy, pandas, scikit-learn>=1.1 y matplotlib. Consulta el README para referencia.

Instalar desde PyPI

Instala la última versión desde PyPI:

pip install sheshe

La aceleración opcional está disponible vía numba:

pip install "sheshe[numba]"

Entorno de desarrollo

Para contribuir a SheShe, instala en modo editable con dependencias de pruebas y ejecuta la suite:

pip install -e ".[dev]"
PYTHONPATH=src pytest -q
