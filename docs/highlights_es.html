Destacados

Características clave

  • Descubre clusters guiados por el modelo mediante ModalBoundaryClustering.
  • Funciona para tareas de clasificación y regresión.
  • Explora subespacios y construye conjuntos con SubspaceScout y ModalScoutEnsemble.
  • Extrae reglas legibles con RegionInterpreter.
  • Busca máximos locales con el método basado en gradiente ShuShu.
  • Calcula fronteras de decisión 2D con CheChe.
  • Visualiza superficies bidimensionales y 3D con utilidades de graficado incorporadas.

Consulta la API rápida para una referencia completa de los objetos públicos.
