Highlights

Key Features

  • Discover model-driven clusters via ModalBoundaryClustering.
  • Works for classification and regression tasks.
  • Explore subspaces and build ensembles with SubspaceScout and ModalScoutEnsemble.
  • Extract human-readable rules using RegionInterpreter.
  • Search for local maxima with gradient-based ShuShu.
  • Compute 2D decision frontiers through CheChe.
  • Visualise pairwise and 3D surfaces with built-in plotting utilities.
